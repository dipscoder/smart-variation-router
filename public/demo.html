This page demonstrates the Smart Variation Router embed script. The script automatically:

  1. Generates or retrieves a unique visitor ID
  2. Assigns the visitor to a variation (A, B, C, or D) using deterministic hashing
  3. Applies the variation by setting data-optim-variation on the HTML element
  4. Tracks the assignment via an image beacon

To use this in your project:

<script src="http://localhost:3000/api/s/YOUR_PROJECT_ID"></script>
